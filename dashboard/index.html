USI formula
Urban Stress Index — original composite metric
USI = ( 0.5 × AQI_norm + 0.3 × Temp_norm + 0.2 × Hum_norm ) × 100
AQI_norm = min(AQI / 300, 1.0) ← WHO hazardous threshold
Temp_norm = clamp((T − 15) / 25, 0, 1) ← 15°C baseline, stress max 40°C
Hum_norm = |H − 50| / 50 ← 50% RH ideal comfort midpoint
